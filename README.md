SiteGroupCms
============
##简介
SiteGroupCms是一套基于.net的网站内容管理系统（CMS），兼顾站群，标准静态页面生成，强大的模板管理语言，详细的开发文档
##功能目录
###1.站点管理    
####1.1站点频道树	
* 1.1.1右键功能	
* 1.1.2 对某频道所有文档的功能	
* **1.2频道管理（栏目管理）**	
* 1.2.1添加栏目	
* 1.2.2 编辑栏目	
* 1.2.3排序栏目

* 1.2.4 共享栏目	
* 1.2.5 预览频道	
* **1.3回收站管理**	
* 1.3.1内容检索

* 1.3.3批量彻底删除	
* 1.3.4清空回收站	
* **1.4审核管理**	
* 1.5完全发布	
* 1.6内容检索	
* 1.7录入统计	
* 1.8站点文件管理
